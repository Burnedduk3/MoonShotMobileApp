# Moon Shot Front End
This project is for moonshot and consist on restaurant-customer manager on real time, 
this app will be just front-end
## yarn install
Install all dependencies for the project.
## yarn start
Start the expo client to let the cellphone app read the QR code and see the application running on the cellphone.
## yarn build-android
Start the process to build the application in the expo server. for android
## yarn build-ios
Start the process to build the application in the expo server. for ios
